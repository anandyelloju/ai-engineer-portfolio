# SQL Practice – sql-practice.com

## Overview

This directory contains SQL queries and exercises completed using **sql-practice.com** as part of my AI Engineering training and self-study.
The goal is to strengthen core database querying skills required for data analysis, machine learning pipelines, and AI systems.

---

## Topics Covered

The exercises focus on fundamental and intermediate SQL concepts including:

* Basic SELECT queries
* Filtering with `WHERE`
* Sorting with `ORDER BY`
* Limiting results with `LIMIT`
* Aggregate functions (`COUNT`, `SUM`, `AVG`, etc.)
* `GROUP BY` and `HAVING`
* Table joins (`INNER JOIN`, `LEFT JOIN`)
* Subqueries
